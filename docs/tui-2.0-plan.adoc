== Key bindings (default chat-first)

| Key | Action |
| Enter | Send message |
| Ctrl-Space / Ctrl-P | Fuzzy palette |
| Ctrl-F / Ctrl-Shift-F | Transcript search (vim off) |
| @ | Attach project file |
| ? / g? | Help overlay (? when input empty) |
| !cmd | Run shell locally (sandboxed), no model turn |
| o/s/d/x | Tool approval (modal) |
| y/n | Gate block proceed/cancel |

Set `COXN_VIM=1` for Normal/Visual/Command modes and vim-style `/` search.
